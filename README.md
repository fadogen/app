# Fadogen

**Open-source PHP development environment for macOS**

A native macOS app providing a complete local development stack with built-in VPS provisioning and deployment. Zero external dependencies — everything is self-contained.

## Features

- **Local Stack** — PHP, PostgreSQL, MySQL, MariaDB, Redis, Valkey, Caddy, Node.js, Bun
- **Automatic HTTPS** — `.localhost` domains with trusted certificates
- **VPS Provisioning** — DigitalOcean, Hetzner, Vultr, Linode
- **Push-to-Deploy** — GitHub Actions with zero-downtime deployments
- **iCloud Sync** — Servers and configurations across devices

## Requirements

- macOS 15.0 (Sequoia) or later
- Apple Silicon
